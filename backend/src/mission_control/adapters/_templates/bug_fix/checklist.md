# Checklist

## Analysis

- [ ] Reproduce the bug with minimal steps
- [ ] Identify root cause and affected components
- [ ] Check similar issues in codebase
- [ ] Assess impact on users and data

## Fix

- [ ] Implement fix following existing patterns
- [ ] Add regression tests to prevent recurrence
- [ ] Update error handling if applicable

## Verification

- [ ] Confirm fix resolves the issue
- [ ] Run full test suite to ensure no breakage
- [ ] Deploy to staging for final validation
