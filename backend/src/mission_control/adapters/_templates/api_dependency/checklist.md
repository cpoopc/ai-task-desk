# Checklist

## Design

- [ ] Define the external API contract (endpoints, request/response schemas)
- [ ] Document rate limiting and quota expectations
- [ ] Identify authentication mechanism (API key, OAuth, JWT, etc.)
- [ ] Map all error codes and their meanings

## Observability

- [ ] Add request/response logging with correlation IDs
- [ ] Set up latency histograms for API calls
- [ ] Create alerts for API errors and timeouts
- [ ] Document retry strategy and backoff configuration
